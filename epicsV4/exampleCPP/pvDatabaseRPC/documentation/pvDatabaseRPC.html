EPICS exampleCPP/pvDatabaseRPC

Release 4.2 - 2016.04.25

Abstract

pvDatabaseRPC provides an example of a pvDatabase PVRecord which also support an RPC service.

For more information about EPICS generally, please refer to the home page of the Experimental Physics and Industrial Control System.

Table of Contents

Overview

The example has a single PVRecord:

mydevice
structure 
    double x
    double y
    time_t timeStamp

The record has channelRPC support that implements a series of moves. Each move sets new values for x and y. A delay occures between each move.

Start Service

mrk> pwd
/home/epicsv4/master/exampleCPP/pvDatabaseRPC
mrk> bin/$EPICS_HOST_ARCH/exampleRPCMain

To start the service as part of a V3IOC:

mrk> pwd
/home/epicsv4/master/exampleCPP/pvDatabaseRPC/iocBoot/exampleRPC
mrk> ../../bin/$EPICS_HOST_ARCH/exampleRPC st.cmd

Start Client

mrk> pwd
/home/epicsv4/master/exampleCPP/pvDatabaseRPC
mrk> bin/$EPICS_HOST_ARCH/move 10 20 30 40

This example will result in two moves. The first will move to x=10 and y=20. The second will move to x=30 and y=40. As many moves as desired can be specified but an even number of arguments must be given.
